ConsultMe

O ConsultMe é uma plataforma desenvolvida para facilitar o gerenciamento de consultas e agendamentos. O foco é oferecer uma interface intuitiva para profissionais e clientes, otimizando o fluxo de atendimento.

Siga estas instruções para colocar o projeto em execução na sua máquina local para fins de desenvolvimento e teste.

Pré-requisitos
Antes de começar, você vai precisar ter instalado em sua máquina:

Git

Node.js ou Docker (dependendo da sua stack principal)

Instalação
Clone o repositório:

Bash
git clone https://github.com/LeoSilvaC/ConsultMe.git
Acesse a pasta do projeto:

Bash
cd ConsultMe
Instale as dependências:

Bash
npm install
Inicie a aplicação:

Bash
npm run dev
Acesso para Teste

Tecnologias Utilizadas:
Frontend: HTML,CSS E JAVASCRIPT

Backend: PYTHON, FLASK

Banco de Dados: SQLALCHEMY

FuncionalidadeS:
- Cadastro de pacientes e profissionais.
- Agendamento em tempo real.
- Painel administrativo para gestão de horários.
- Histórico de consultas.
